# Contact form task

### Overview:

```
As a frontend developer, your task is to build a contact form
using Vue.js 3, TypeScript, and a state management system
such as Vuex.
The form should have a clean and responsive design,
and should allow the user to perform CRUD
(Create, Read, Update, Delete) operations on contact records.
All form submissions and data retrieval should be done securely using HTTPS.
```

### Requirements:

```
The contact form should have the following fields: Name, Email, Phone, and Message.
All fields are required, and should have appropriate validation (e.g. email validation).
The form should have a clear and user-friendly design, with appropriate error messages for invalid input.
The form should be integrated with Vuex to manage the form's state and handle CRUD operations on contact records.
The form should submit user input securely using HTTPS.
The form should have the following features:
Create: Ability to create a new contact record.
Read: Ability to display a list of all contact records.
Update: Ability to edit an existing contact record.
Delete: Ability to delete an existing contact record.
Form Validation: Proper validation should be applied to all fields.
All data retrieval and form submissions should be done securely using HTTPS.
```


### Evaluation Criteria:

```
Understanding of Vue.js 3, TypeScript, Vuex, HTTPS, form validation, and design principles.
Ability to create a clean and user-friendly form with appropriate validation and error handling.
Ability to integrate the form with Vuex to manage the form's state and handle CRUD operations.
Ability to securely submit user input using HTTPS.
Ability to implement proper form validation for all fields.
Ability to implement CRUD operations (Create, Read, Update, Delete) for contact records.
```
